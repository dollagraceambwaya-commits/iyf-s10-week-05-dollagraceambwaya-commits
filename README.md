# iyf-s10-week-05-dollagraceambwaya-commits

## To-Do List (DOM Manipulation)

## Overview
This project is part of **Season 10 – Week 5 DOM Manipulation** at IYF Weekend Academy.  
It demonstrates how to use **JavaScript DOM methods** to create, update, and delete elements dynamically in a web application.

The app allows users to:
- Add new tasks
- Mark tasks as completed
- Filter tasks (All, Active, Completed)
- Edit tasks by double-clicking
- Delete tasks
- Clear all completed tasks
- View the count of active tasks



## Features
- **Dynamic rendering** of tasks using `renderTodos()`
- **Event delegation** for handling clicks on list items
- **Filter buttons** with active state styling
- **Stats section** showing items left
- **Edit mode** with Enter (save) and Escape (cancel)



## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation, Event Listeners, Array Methods)

## How to Run
1. Open `index.html` in your browser.  
2. Type a task in the input field and click **Add**.  
3. Use the filter buttons to view **All**, **Active**, or **Completed** tasks.
